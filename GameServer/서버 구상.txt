플레이어 이동을 구현하기 위해서
- 길찾기 전용 유니티 프로젝트(길찾기 서버)를 만든다.
- 클라이언트에 들어갈 Ground와 똑같은 3D object를 길찾기 서버에 올리고 NavMesh Bake.
- 길찾기 서버는 메인 게임서버와 통신하며
- 길찾기 서버가 서버를 메인게임서버가 클라를 맡는다.
(서버 켜는 순서 : 로그인 - 채팅 - 길찾기 - 게임서버 )

플레이어 이동 로직
1. 클라이언트 이동 요청 -> 게임서버
2. 게임서버 이동 요청 -> 길찾기서버
3. 길찾기서버 이동 처리
4. 길찾기서버 이동 응답 -> 게임서버
5. 게임서버 이동 응답 -> 클라이언트

길찾기 서버는 임시로 부르는 이름으로
차후에 공격 등 게임로직에 관련된 서버로 이용할 것
